# HTML-exercise

这是一个简单的HTML静态页面，还原了携程2025全球100口碑榜的部分细节，增加部分CSS布局与字体等样式。整个代码基于HTML5语义化原则，且添加了移动端适配，允许导航栏自动调整宽度。
