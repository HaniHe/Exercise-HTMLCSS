# HTML+CSS-exercise

这是一个简单的HTML静态页面，还原了携程2025全球100口碑榜的部分细节，增加CSS布局与字体等样式。整个代码基于HTML5语义化原则，且添加了移动端适配，允许导航栏自动调整宽度。整个项目用于HTML与CSS的学习与练习。

## 项目功能
1. 整个网页结构包含头部、内容区和底部
2. 在页面滚动时，头部内容始终保持吸顶状态
3. 内容区均分为两列
4. 响应式布局，当屏幕宽度小于600px时，两列布局变为单列布局
